# 🚀 Quick Start - 2 Minutes

## Prerequisites
- Node.js 16+
- Python 3.8+
- MySQL on port 3308 (running)

## Start Everything
```bash
start_all.bat
```

Wait for all 3 windows to open (5-10 seconds).

## Login
```
URL: http://localhost:5173
Email: admin@choco.com
Password: password123
```

## You're Ready! ✅

**API:** http://localhost:5000/api  
**Backend:** Running on port 5000  
**Frontend:** Running on port 5173  
**Vision:** Running on port 8001

---

## If Something Breaks

### Backend offline?
```bash
cd backend
npm run dev
```

### Database error?
```bash
cd backend
npm run init-db
```
